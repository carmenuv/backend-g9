pip install NOM_LIBRERIA > instalar la libreria, si no la encuentra lanzará un error
python -m install --upgrade pip > actualizar pip(gestor de paquetes) a la última versión
pip freeze > lista todas las librerías instaladas en el scope actualizar
pip list > lista TODAS las librerías (instaladas manualmente y las que tiene por defecto python)
