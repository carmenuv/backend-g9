flask db init => Inicializar las migraciones en mi proyecto (debemos tener instalado flask-migrate)
flask db migrate -m "mensaje" => Crea una nueva migración con el nombre que le colocamos
flask db upgrade => ejecuta todas las migraciones pendientes en la base de datos
flask db downgrade => ejecutará un revert de todas las migraciones que hemos creado hasta el momento NOTA: NO REALIZAR ESTO PORQUE SINO PERDEREMOS TODA LA INFORMACIÓN DE LA BD
flask db downgrade --tag ID_MIGRACION => hace el revert PERO SOLAMENTE hsta la migracion que nosotros deseamos
